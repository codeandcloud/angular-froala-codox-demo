# Angular Codox Demo

## Running Application

Run `npm install` to install packages.  
Run `ng serve` for a dev server.

Open the application in two tabs

1. [http://localhost:4200/alice](http://localhost:4200/alice)
2. [http://localhost:4200/bob](http://localhost:4200/bob)
